# Smart Complaint Categorization System

An AI-powered complaint management solution designed to automate complaint classification, priority prediction, sentiment analysis, and department routing using Natural Language Processing (NLP) and Machine Learning techniques.

## Features
Automated complaint categorization
  Priority prediction
  Sentiment analysis
  Smart department routing
 Complaint analytics dashboard
  Escalation management
